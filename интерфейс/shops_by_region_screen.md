# 🌍 Shops by Region Screen (Экран Магазинов по Регионам)

## 📋 Описание
Геолокационная карта местных магазинов с фильтрами расстояния, категориями и возможностью открытия региональных маркетплейсов.

## 🔧 Ключевые особенности
- **Location-based discovery**: Поиск магазинов по геолокации
- **Distance filtering**: Настраиваемые радиусы поиска
- **Category filtering**: Фильтрация по типам магазинов
- **Map integration**: Интерактивная карта с маркерами
- **Regional marketplace**: Поддержка местных экономик
