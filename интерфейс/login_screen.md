# 🔐 LOGIN SCREEN - Экран входа в Spark

## 📱 Общая структура
```
╭─────────────────────────────────────╮
│                                     │ ← Пустой AppBar
├─────────────────────────────────────┤
│                                     │
│            🌟                        │ ← Центральный элемент
│                                     │
│        SPARK                        │
│                                     │
│ ╭─────────────────────────────────╮ │ ← Email поле
│ │ 📧 Введите email                 │ │
│ ╰─────────────────────────────────╯ │
│                                     │
│ ╭─────────────────────────────────╮ │ ← Password поле
│ │ 🔒 Пароль                        │ │
│ ╰─────────────────────────────────╯ │
│                                     │
│ ╭────── Войти ────────────────────╮ │ ← Primary кнопка
│ ╰─────────────────────────────────╯ │
│                                     │
│     Забыли пароль? Нет аккаунта?     │ ← Ссылки
│                                     │
╰─────────────────────────────────────╯
```

## 🎨 Визуальные принципы

### 🌟 Централизованный дизайн
- **Главный элемент**: Spark логотип в центре экрана
- **Белый фон**: Чистый, минималистичный background
- **Пространство**: Много воздуха вокруг элементов

### 📝 Форма входа
- ** карточка формы**: Субтильный border radius 16px
- **Отступы**: 48px padding сверху и снизу
- **Поля ввода**: Border radius 12px, без outline
- **Иконки**: В каждом поле (email/lock)

### 🔘 Кнопки
- **Primary button**: Spark красный цвет #FF0000
- **Вторичные ссылки**: Синий цвет #0066CC
- **Underlines**: Для кликабельных текстов

## 📱 Адаптивность

### Для больших экранов:
- **Максимальная ширина формы**: 400px
- **Центрирование**: Через MediaQuery и padding

### Для маленьких экранов:
- **Относительные отступы**: % от screen height
- **Шрифты**: Масштабирование по screen width

## 🔄 Анимации и состояния

### Загрузка входа:
- **Button text**: "Входим..." с CircularProgressIndicator
- **Отключение**: Все поля и кнопки disabled

### Ошибки:
- **Border color**: Красный для поля с ошибкой
- **Helper text**: Красный текст под полем

### Успех:
- **Фейд аут**: Плавное исчезновение формы
- **Переход**: В основной фид

Все изменения интерфейса экрана входа согласовываются с вами! ✨
